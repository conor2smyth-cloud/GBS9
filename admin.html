Manage Tonight’s Event

To update tonight’s event:

  1. Open data/settings.json in GitHub
  2. Edit the eventCode, eventName, and eventDescription
  3. Commit your changes → they’ll be live immediately

Guests will need the updated passcode on tonights-event.html to unlock details.

Data Flow Reference

This diagram shows how cocktails, beers, and equipment flow from Excel into the site:

  1. Edit the Excel file with cocktails, beers, and equipment.
  2. Export to cocktails.json in /data.
  3. Upload matching images to /images/cocktails, /images/beer, or /images/equipment.
  4. The website auto-populates based on cocktails.json.
