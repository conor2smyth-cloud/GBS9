Image Pipeline

Flow process:

  1. Upload images → /images/_raw/cocktails, /beer, or /equipment
  2. Run python process_images.py
  3. Outputs resized + watermarked images → /images
  4. Updates data/images.json
  5. Frontend pulls automatically from JSON
