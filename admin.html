Cocktail Data Flow

This diagram shows how cocktail data and images move through the system.

  • 1. Upload Images → Place raw cocktail/equipment images into /images/_raw/.
  • 2. Process → The Python script resizes, watermarks, and moves them to /images/cocktails or /images/equipment.
  • 3. JSON Update → Cocktails are added/edited in /data/cocktails.json.
  • 4. Frontend Display → Premium Listings, Tinder Menu, and Lookbook pull dynamically from JSON + processed images.
