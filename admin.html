Admin Dashboard

Here’s where you manage everything behind the bar:

1. Upload New Images

Add cocktail, beer, or equipment photos to images/_raw. They’ll be processed automatically.

2. Update Cocktails

Edit the master file at data/cocktails.json. Each entry includes:

  • Name (Cocktail name)
  • Base (e.g. Vodka, Whiskey)
  • Glass (Martini, Coupe, Old Fashioned)
  • Ingredients (Array of strings)
  • Short (Brief blurb, also used for Tinder bios)

3. Import from Excel

You can prepare a cocktail list in Excel. We’ll provide a script that automatically converts Excel into JSON for data/cocktails.json.

4. Image Pipeline Flow

Reference diagram: Upload to _raw → Processing (resize, watermark, tint) → Final images in /cocktails, /beer, /equipment.
