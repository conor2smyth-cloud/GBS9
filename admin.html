Admin Panel

Welcome to the admin section. Use the tools below to update cocktails, images, and listings.

Image Pipeline

Drop new images into images/_raw/ and run update_site.py to process them automatically.

Excel Import

Upload your Excel file of drinks to data/. Then run generate_json.py to refresh cocktails.json.

Flow Reference

  1. Add raw images → images/_raw/
  2. Run update_site.py
  3. Prepare Excel list of cocktails
  4. Run generate_json.py
  5. Commit & push to GitHub
